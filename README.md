# 🎉 EventFinder (Nepvent)

A modern web application to discover, create, and manage local events in your area.  
Built for communities to connect through real-world experiences.

---

## 🚀 Live Demo
https://nepvent-kd8q.vercel.app/

---

## ✨ Features

- 🔍 Discover nearby events
- ➕ Create and publish events
- 👤 User authentication
- 📍 Location-based event browsing
- 🗓️ Event scheduling system
- ⚡ Fast and responsive UI
- 🌙 Modern UI with dark/light support

---

## 🛠️ Tech Stack

- **Frontend:** Next.js (App Router)
- **Styling:** Tailwind CSS
- **Backend:** Supabase
- **Database:** PostgreSQL (via Supabase)
- **Auth:** Supabase Auth
- **Deployment:** Vercel

---

## 📁 Project Structure
app/ → Pages (App Router)
components/ → Reusable UI components
lib/ → Supabase client + helpers
hooks/ → Custom React hooks
public/ → Static assets
styles/ → Global styles

## ⚙️ Environment Variables

Create a `.env.local` file:

```env

NEXT_PUBLIC_SUPABASE_ANON_KEY=your_supabase_anon_key
